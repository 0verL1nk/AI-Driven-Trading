# AI Trading Monitor - Frontend

黑白简约风格的实时交易监控界面

## 快速启动

```bash
# 安装依赖
npm install

# 开发模式运行
npm run dev

# 访问
http://localhost:3000
```

## 功能

- 📊 实时账户价值图表
- 💰 币种价格实时显示
- 🤖 AI决策历史记录
- 📈 当前持仓展示
- 🎨 黑白简约UI设计

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts（图表库）
- date-fns（时间格式化）

## API配置

确保后端API服务运行在 `http://localhost:8000`
